# API ROUTES

## GET methods

### movie Entity
- All movies
- Movie by movieId
- Movie by title

### watchedMovie Entity
- All watched movies
- Watched Movies by Id (composite)
- WatchedMovie by username
- WatchedMovie by movie

### review Entity
- All reviews
- Reviews by Id (composite)
- Review by username

### user Entity
- All users
- Users by Id
- Users by review

## POST methods

### movie Entity
- New Movie

### watchedMovie Entity
- New watched movie entries

### review Entity
- New review
  
### user Entity
- Create new user
  
## PUT methods

### movie Entity
- Update Movie

### watchedMovie Entity
- Update watched movie status (Watched Date)

### review Entity
- Update review

### user Entity
- Update user info
